# SAB Campus Excel Analyzer

## How to Run on Another PC

### Step 1: Install Python
1. Download Python 3.9+ from: https://www.python.org/downloads/
2. **IMPORTANT**: Check âœ… "Add Python to PATH" during installation

### Step 2: Copy Files
Copy the entire `examtool` folder to the other PC (excluding the `.venv` folder to save space)

### Step 3: First Time Setup
1. Double-click **`SETUP_FIRST.bat`**
2. Wait for installation to complete (may take 2-3 minutes)

### Step 4: Run the App
1. Double-click **`Run_Excel_Analyzer.bat`**
2. The browser will open automatically

---

## Files to Copy (Minimum Required)
- `app.py`
- `requirements.txt`
- `SETUP_FIRST.bat`
- `Run_Excel_Analyzer.bat`
- `README.md`

## Troubleshooting
- If you get "Python not found" error, reinstall Python with "Add to PATH" checked
- If the browser doesn't open, manually go to: http://localhost:8501
